# TaoTeKing
---
auteur: cagliostro atfield2501@gmail.com

Oracle Taoïste basé sur les 64 héxagrammes

Aucune dépendance 
---
## Instalation 4 Linux
``` shell
git clone https://github.com/ATField2501/TaoTeKing.git
cd TaoTeKing
sudo chmod +x CaglioOracle.py
```
---

  Caglio-Oracle -o (oracle) * (verbeux)
  Caglio-Oracle -t (lister 8 trigrammes)
  Caglio-Oracle -H 12 (Afficher hexagramme n°12)
  Caglio-Oracle -r (Rivière du Tao) * (gui)
